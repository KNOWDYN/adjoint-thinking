# Adjoint Thinking Scenario Companion Site v1.0.0

This package is designed for immediate GitHub Pages deployment from `/docs`.

## Deploy

1. Extract this ZIP.
2. Copy the `docs/` folder into the root of your GitHub repository.
3. Commit and push.
4. In repository settings, open Pages.
5. Select deployment from branch, choose your main branch, and choose `/docs` as the publishing folder.
6. Save and wait for GitHub Pages to publish.

## Included

- Prebuilt static site in `docs/`.
- Editable Markdown source vault in `source/quartz-content-vault.zip`.
- Environment Pack v1.0.0 in `docs/downloads/`.

## Reality note

This is a prebuilt Quartz-style static site for no-code deployment. Editing the source vault later requires rebuilding the site or requesting a new generated release.
