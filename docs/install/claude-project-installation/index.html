Install in Claude Projects

Install the Adjoint Thinking Environment Pack in Claude Projects without code, APIs, or developer tooling.

  1. Create a new Project in Claude.
  2. Use the short Claude instruction block from the v1.0 pack.
  3. Add the router, compact prompt assets, unified response template, and validation prompts to Project knowledge.
  4. Start a new chat inside the Project and run the validation prompt.
  5. Complete the setup interview.
  6. Keep detailed prompt assets in knowledge rather than overloading the instruction field.
