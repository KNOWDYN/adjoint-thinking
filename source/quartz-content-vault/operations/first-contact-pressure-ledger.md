# First Contact Pressure Ledger

Protects the first human encounter with the work before AI frames it.

## Trigger

Use when the user has not yet faced the material, question, source, draft, or problem directly.

## Hard gate

No machine answer should become the frame until the user has made one human mark: question, doubt, sketch, claim, or pressure sentence.
