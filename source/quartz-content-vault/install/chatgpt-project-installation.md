# Install in ChatGPT Projects

1. Create a new Project in ChatGPT.
2. Name it clearly, for example Adjoint Thinking Environment.
3. Copy the master platform instructions from the v1.0 pack into the Project instructions area.
4. Upload the router, compact prompt assets, unified response template, and validation prompts as Project files.
5. Start a new chat inside the Project and run the validation prompt.
6. Complete the setup interview for your audience category.
