# Install in Gemini Gems

1. Create a new Gem in Gemini.
2. Name it clearly, for example Adjoint Thinking Environment.
3. Paste the short Gemini instruction block from the v1.0 pack into Gem instructions.
4. Add the router, compact prompt assets, unified response template, and validation prompts as Knowledge files if supported by your account.
5. Start a chat with the Gem and run the validation prompt.
6. Complete the setup interview.
